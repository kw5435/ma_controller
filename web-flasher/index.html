Probleme beim Verbinden?
Kein Port sichtbar
CH340-Treiber installieren: Windows · macOS (Linux: meist integriert)
Flash schlägt fehl
BOOT-Taster auf dem CYD gedrückt halten, dann erst Flash klicken. Loslassen wenn "Connecting..." erscheint.
Manuell per esptool flashen
pip install esptool
esptool.py --port COM3 --baud 460800 write_flash 0x0 firmware-merged.bin
Vollständiges Löschen + Neu-Flash
esptool.py --port COM3 erase_flash
esptool.py --port COM3 write_flash 0x0 firmware-merged.bin
